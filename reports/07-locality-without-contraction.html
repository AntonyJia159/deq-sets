Locality Without Contraction: A Susceptibility Theory of Maintainable Equilibria

Zhibai Jia · 27 June 2026 · consolidates the locality proof and the conditioning experiments

What this note is. Note #6 bet that contraction, not linearity, is what makes an equilibrium graph model cheaply maintainable under edits. Chasing that down forced a sharper question — exactly what property licenses local maintenance — and the answer is not contraction. This note lays out the locality argument as a clean logical spine, replaces the spectral-radius story with the right object ($\sigma_{\min}(I-J)$), and reports the experiments that confirm it. Honesty up front: the mathematics here is classical numerical linear algebra (Demko–Moss–Smith; Benzi–Golub; Faber/Chebyshev approximation; pseudospectra). Our candidate contribution is the bridge — using it as the well-posedness-and-maintenance criterion for (possibly non-contractive) equilibrium models — and that novelty is still pending a proper search.

1. The question: when is an edit a local event?

A graph equilibrium model holds a state $z^\star$ solving $z^\star=f(z^\star)$, where $f$ does one round of (possibly signed, attention-based, nonlinear) message passing with frozen weights. Maintenance is: the graph is edited (a node/edge deleted), and we want the new equilibrium without recomputing from scratch — ideally by touching only nodes near the edit. That is possible only if the edit's influence decays with graph distance. Everything below is a derivation of when, and how fast, it does.

  1. Poke the fixed point → the response solves a linear system in $(I-J)$; the influence map is the resolvent $(I-J)^{-1}$.
  2. A node deletion is a compact poke (support on an $O(1)$-hop ball), so the response is the resolvent's own decay profile.
  3. $(I-J)$ is graph-sparse; a low-degree polynomial in it reaches only a few hops.
  4. The inverse is well-approximated by such a polynomial (Chebyshev / Faber), with error set by the spectrum's distance from the pole $\mu=+1$.
  5. Demko–Moss–Smith / Benzi–Golub: therefore the resolvent entries decay exponentially in graph geodesic distance.
  6. The governing quantity is $\sigma_{\min}(I-J)$ (distance of the spectrum from $+1$), not $\rho(J)$ — so the maintainable regime is strictly larger than contraction.

2. Susceptibility: poking the equilibrium

Suppose we are at equilibrium and add a small static probe $\delta f$ to the system, so the condition becomes $u = f(u) + \delta f$. The state shifts $u^\star \to u^\star + \delta u$. First-order Taylor:

$$u^\star + \delta u = f(u^\star) + J_f(u^\star)\,\delta u + \delta f.$$

Because $u^\star = f(u^\star)$, those terms cancel and we are left with a strictly linear system for the response:

$$\big(I - J_f(u^\star)\big)\,\delta u = \delta f \qquad\Longrightarrow\qquad \delta u = \big(I - J_f\big)^{-1}\,\delta f.$$

This is the classic $L\,u=b$ architecture with $L = I - J_f(u^\star)$, the static linear operator at the equilibrium. The inverse $(I-J_f)^{-1}$ is the resolvent (physicists: the Green's function): its entry $(I-J_f)^{-1}_{ie}$ answers "if I nudge node $e$, how much does node $i$ move?" Edit-locality is exactly the statement that these entries are tiny for far $i$.

3. A node deletion is a compact poke

The probe $\delta f$ need not be a literal external force — any local change to the equations acts as one. Deleting node $v$ changes the update map $f\to f'$. Measure how wrong the old solution is under the new rules: $r_i = f'_i(z^\star) - z^\star_i$. For any node $i$ whose neighborhood is untouched, $f'_i=f_i$ and so $r_i = f_i(z^\star)-z^\star_i = 0$. The residual is nonzero only where the rule actually changed — the $1$-hop neighbors of $v$ (or $2$-hop, once symmetric degree re-normalization is included). So $r$ has sharp, compact support: a lump on an $O(1)$-hop ball, exactly zero outside.

Why compactness matters. The response is $\delta u = (I-J)^{-1} r = \sum_{e\in\text{supp}(r)} (I-J)^{-1}_{\cdot e}\, r_e$. Convolving a compactly-supported lump with the resolvent's decaying kernel returns the kernel's own decay rate — the finite support only inflates the prefactor. (If the edit's footprint were itself an extended, slowly-decaying tail, the two decays would convolve and the slower one would bottleneck locality. Node/edge deletion avoids this because its footprint is compact.) The magnitude of the deletion lives in $\lVert r\rVert$; the decay rate is a property of $(I-J)^{-1}$ alone.

4. The engine: a low-degree polynomial in a sparse matrix is short-range

$J$ does one hop of message passing, so $J_{ij}\neq 0$ only on edges — $(I-J)$ is graph-sparse (graph-bandwidth $1$). Now look at powers:

$$(J^k)_{ij} = \!\!\sum_{\text{length-}k\text{ walks } j\to i}\!\! \prod (\text{edge gains}) \;=\; 0 \quad\text{whenever } d(i,j) > k.$$

$J^0$ reaches $0$ hops, $J^1$ reaches $1$, $J^k$ reaches $k$. Hence any degree-$k$ polynomial $p_k(J)=\sum_{l\le k} c_l J^l$ has $(p_k(J))_{ij}=0$ for $d(i,j)>k$: a low-degree polynomial in $J$ is automatically short-range on the graph. This is the bridge between algebra (polynomial degree) and geometry (graph distance).

5. The inverse is such a polynomial — approximately (Chebyshev / Faber)

We want $(I-J)^{-1}=g(J)$ with $g(\mu)=\dfrac{1}{1-\mu}$. If a degree-$k$ polynomial $p_k$ approximates $g$ well on the spectrum of $J$, then $(I-J)^{-1}\approx p_k(J)$ is short-range, and the far-field entries are bounded by the approximation error:

$$\big|(I-J)^{-1}_{ij}\big| = \big|(g(J)-p_k(J))_{ij}\big| \le \max_{\mu\in\sigma(J)} |g(\mu)-p_k(\mu)| \quad\text{for } d(i,j)>k.$$

So the entry at distance $d$ is controlled by how well a degree-$d$ polynomial fits $1/(1-\mu)$ over the eigenvalues. Which polynomials, and how fast do they converge?

  • Taylor (the Neumann series $\sum_k\mu^k$): fits near a single point $\mu=0$; converges only inside $|\mu|<1$ — the contraction case.
  • Chebyshev: the best uniform fit over a real interval $[a,b]$; converges geometrically at a rate set by the interval's distance to the singularity of $g$ — even if part of $[a,b]$ lies outside the unit disk.
  • Faber: the complex-plane generalization — best polynomial fit over a region enclosing the spectrum — needed when $J$ is non-symmetric and its eigenvalues are complex (our case).

The one pole of $g$ sits at $\mu=+1$. So convergence — and hence spatial decay — is governed by how far the spectrum stays from $+1$, not by whether it sits inside the unit disk.

6. The conclusion: exponential decay on graph geodesics

This is exactly the setting of Demko, Moss & Smith (1984) for inverses of banded matrices, generalized to graph-sparse matrices by Benzi & Golub and collaborators: for $(I-J)$ sparse w.r.t. the graph and well-conditioned,

$$\boxed{\;\big|(I-J)^{-1}_{ij}\big| \;\le\; C\, q^{\,d(i,j)}, \qquad q = \frac{\sqrt{\kappa}-1}{\sqrt{\kappa}+1},\quad \kappa=\kappa(I-J).\;}$$

The entries decay exponentially in graph geodesic distance, with a rate set entirely by the condition number of $(I-J)$ — dimension-independent. Combined with §3 (the edit is a compact poke), the deletion response decays at this same rate: maintenance is local.

Contraction is the comfortable special case. If $\lVert J\rVert<1$, the Taylor/Neumann series $\sum_k J^k$ already converges and gives decay rate $q=\lVert J\rVert$. But the Chebyshev/Faber route shows this is sufficient, not necessary: the inverse stays short-range whenever the spectrum avoids $+1$, even if $\rho(J)>1$ and the Neumann series diverges. Locality is a property of conditioning, not of contraction.

7. The right probe: $\sigma_{\min}(I-J)$, not $\rho(J)$, not the eigenvalue gap

Since $J$ is non-symmetric (signed attention is order-dependent; value/output maps and nonlinearities break symmetry), its eigenvalues are complex and — crucially — misleading. The exact distance-to-singularity is the smallest singular value, $\sigma_{\min}(I-J) = 1/\lVert (I-J)^{-1}\rVert$. For any eigenvalue $\lambda$ with eigenvector $v$, $\lVert (I-J)v\rVert/\lVert v\rVert = |1-\lambda|$, and since $\sigma_{\min}$ is the minimum over all directions,

$$\mathrm{dist}\big(1, W(J)\big) \;\le\; \sigma_{\min}(I-J) \;\le\; \min_i |1-\lambda_i|, \qquad\text{equality on the right iff } J \text{ is normal.}$$

The eigenvalue gap $\min_i|1-\lambda_i|$ is an over-optimistic upper bound; for non-normal $J$ the resolvent is large throughout a neighborhood of the spectrum (the pseudospectrum), and the honest margin is the distance from $+1$ to the field of values $W(J)$, which $\sigma_{\min}(I-J)$ tracks. Practically: report $\sigma_{\min}(I-J)$. The spectral radius $\rho(J)$ is doubly wrong — $+1$ is the danger point, not the unit circle, and eigenvalues do not even capture non-normal behavior.

8. Two thresholds: existence vs. practical locality

"Low-degree polynomial" is relative to the screening length. With $q=(\sqrt\kappa-1)/(\sqrt\kappa+1)$,

$$\xi = \frac{-1}{\ln q} \;\approx\; \tfrac{1}{2}\sqrt{\kappa(I-J)}\,,$$

so the effective hop-reach of the resolvent grows like $\sqrt{\kappa}$. This separates two conditions that are easy to conflate:

  • Existence / well-posedness: $\sigma_{\min}(I-J)>0$ — the fixed point is unique-ish and findable (by Broyden). But the response can still be delocalized if $\kappa$ is huge.
  • Practical locality / cheap maintenance: $\kappa(I-J)$ small enough that $\tfrac12\sqrt\kappa \ll$ graph diameter. Only then is the resolvent sharply local.
"Local in name only" is a real failure mode. An equilibrium can sit just barely off $\mu=+1$ (large $\kappa$): it exists and Broyden finds it, but $\xi$ exceeds the graph diameter and the resolvent is effectively dense — maintenance is not cheap. Two consequences: (i) the Demko/Faber bound is pessimistic, so don't compute $\xi$ from it — measure the actual edit-response decay; (ii) you need a high-diameter graph (e.g. roman-empire) for locality to be observable at all. The useful non-contractive regime is the well-conditioned one: expansive eigenvalues that are large in magnitude but far from $+1$ (negative/complex), giving small $\kappa$ despite $\rho>1$.

9b. Broyden reaches non-contractive fixed points — and a long-standing bug

Every prior attempt of ours to use quasi-Newton solvers (Anderson, Broyden) on graph DEQs had failed (stagnation / NaN). The cause was mundane and total: TorchDEQ's solvers treat tensor dimension $0$ as a batch, so a graph state of shape $(N,d)$ was solved as $N$ independent per-node systems — Broyden's inverse-Jacobian became block-diagonal and could not see cross-node coupling. Presenting a single graph as $(1, N, d)$ fixes it. With that correction, on a synthetic grid:

$\rho(J)$ regimePicardAndersonBroyden
$\rho<1$ (contractive)~1e−8~1e−8~1e−8
$\rho=1.13$–$1.23$ (non-contractive, well-conditioned)diverges~1e−6~1e−7

So genuine, well-conditioned fixed points with $\rho(J)>1$ exist and are findable — they are simply invisible to Picard. This is the empirical foundation for maintenance beyond contraction.

9d. On roman: a real non-contractive equilibrium, and it is edit-local

The synthetic results above are confirmed on the real model. We train the soft cell (free weights, LayerNorm inside, light jac_reg for stability) on roman-empire and solve it with Broyden, presenting the graph as $(1,N,d)$ so the solver sees one coupled system. It trains end-to-end for 120 epochs to test $0.677$ with $\rho(J)\approx 1.2$–$1.6$ throughout and Broyden residual $\sim10^{-3}$ every step — a genuine, stable, non-contractive equilibrium, the regime Picard cannot enter.

Accuracy is not the claim. The real non-contractive equilibrium ($0.677$) is comparable to — not better than — the contractive cell ($0.687$). The earlier $0.716$ from the $s_{\max}$ sweep was a Picard-truncation artifact (a finite-depth deep net), not the accuracy of any fixed point. On roman the useful graph signal beyond $\sim0.68$ is long-range, which a screened equilibrium deliberately does not chase. The point of this run is maintainability, not accuracy.

Edit-locality at that fixed point: delete a node, warm-start Broyden from the pre-edit $z^\star$, and measure the response $\lVert\Delta_i\rVert$ versus BFS distance from the edit. The near field is sharply local — $1.09 \to 7\times10^{-4}$ over five hops, near-field $\xi \approx 0.5$–$0.7$ hop even at $\rho=1.3$. Beyond that sits a non-decaying floor at $\sim10^{-2}$, which turns out to equal the warm-vs-cold basin gap: not a long-range edit response, but the global ambiguity of which low-residual point the solver lands on — the signature of a single near-critical mode (small $\sigma_{\min}(I-J)$).

9e. The conditioning floor is benign

Is that floor a problem? A four-part verification (non-contractive model A at $\rho=1.39$ vs a strongly-regularized contrast model B) says no:

  • Zero far-field prediction flips. Comparing warm-start re-solve to a cold full recompute, the fraction of nodes whose predicted label changes beyond the edit is $0.0000$ for every target. The latent floor is task-irrelevant; maintenance is label-exact everywhere except the genuinely-affected region.
  • Conditioning, not $\rho$, controls it — and it is removable. Model B lands at $\rho=1.006$ (still at the edge) yet shows no floor (basin gap $\sim10^{-7}$), purely because stronger jac_reg kept $\sigma_{\min}(I-J)$ healthy. The floor is a knob, not a wall.
  • One fixed, delocalized mode. The warm$-$cold difference vector has cosine $0.69$ across different deletions (one intrinsic $v_{\min}$ of $(I-J)$, gauge-like), and $\texttt{gap\_near} \gg \texttt{gap\_far}$ (e.g. $0.54$ vs $7.6\times10^{-6}$): the real response is local; the floor is a low-level global mode.
  • Partly solver slop. Tightening the Broyden budget shrinks the gap on about half the targets ($1.1\times10^{-2}\to1.4\times10^{-3}$), so some of the (already small) floor is just under-convergence.
Two regimes, made precise. Contraction gives latent-global-exact maintenance (unique fixed point; warm $\equiv$ cold to machine precision). A well-conditioned non-contractive equilibrium gives label-exact, sharply local maintenance, trading latent-exactness for a benign, $\sigma_{\min}$-controlled floor (a single delocalized mode that changes no predictions and shrinks under regularization). The broadened thesis holds in the practical sense that matters.

10. Where this leaves the thesis

The maintainability we want is governed by a single, measurable quantity — the conditioning of $(I-J)$, equivalently the distance of $J$'s spectrum from $+1$ — and contraction ($\rho<1$) is only a sufficient corner of that condition. This broadens the program from "force a Banach contraction" to "keep $(I-J)$ well-conditioned," which is the regime real DEQs already live in (Broyden, not Picard). Three notions collapse to one boundary: maintainable (resolvent local), well-posed (unique, non-marginal fixed point), and quasi-Newton-solvable ($(I-J)$ nonsingular) all fail together exactly when an eigenvalue of $J$ reaches $+1$.

What is borrowed vs. candidate-new. Borrowed and cited: Green's functions; decay of inverses of sparse/banded well-conditioned matrices (Demko–Moss–Smith 1984; Benzi–Golub, Benzi–Boito–Razouk); Chebyshev/Faber approximation; pseudospectra and the field of values (Trefethen–Embree); the physics of "nearsightedness" (Kohn 1996). Candidate contribution, pending a real novelty check: using $\sigma_{\min}(I-J)$ / field-of-values as the well-posedness-and-maintenance criterion for equilibrium graph models, the resulting characterization of a maintainable non-contractive regime, and the empirics above. The DEQ literature to date stays on $\rho(J)$, contraction, and Frobenius Jacobian regularization (for training stability, not edit-locality) — so the bridge looks open, but that must be verified before it is claimed.

11. Next

Done since the first draft (§9d–9e): the soft cell + Broyden on roman is a real non-contractive equilibrium, it is sharply edit-local, and the conditioning floor is benign (label-exact maintenance). What remains:

  1. A more favorable showcase task. roman is adversarial to us: its useful graph signal is long-range, so accuracy beyond $\sim0.68$ requires the long reach that destroys locality. We want a task that is beyond-linear and local and maintenance-natural (e.g. grid-structured Minesweeper, or a designed local-nonlinear / NCA-regeneration task), where both halves — expressiveness over linear and cheap local maintenance — show at once.
  2. Novelty check: Semantic Scholar + human gut-check on whether the $\sigma_{\min}(I-J)$ / field-of-values bridge to implicit-model maintenance is genuinely unoccupied.
  3. $\sigma_{\min}(I-J)$ at scale (matrix-free, deferred): correlate the measured screening length and floor with the conditioning directly.
